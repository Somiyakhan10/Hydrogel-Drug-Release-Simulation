# Hydrogel Drug Release Simulation

## Project Description
This project simulates and analyzes **drug release from hydrogel-based biomaterials** using experimental datasets and diffusion-based mathematical models.  
It is designed for **biomaterials and biomedical engineering applications**, including controlled drug delivery and regenerative medicine research.

The repository includes:

- Data: Real or synthetic hydrogel drug release datasets.
- Analysis Notebook: A Google Colab-ready notebook with visualization and model fitting.
- Scripts: Python scripts for drug release models (Higuchi & Korsmeyer-Peppas).
- Outputs: CSV files of fitted model predictions.

---

## Features

1. Upload experimental drug release datasets (CSV or Excel).
2. Plot experimental drug release curves for multiple hydrogel samples.
3. Fit **Higuchi** and **Korsmeyer-Peppas** models to data.
4. Generate time-dependent drug release curves.
5. Save fitted model predictions for further analysis.
